# 经费卡财务决算工具

这是一个用于处理北京科技大学校团委办公室财务组经费卡财务决算 Excel 报表的自动化 Python 脚本。该工具可以读取导出的原始经费明细表，自动建立各项费用分类标签页，并根据科目代码将每一笔支出精准分流。最后会在各分类表底部自动生成求和公式，并输出一张全局的财务数据汇总表。



## 核心功能

* 自动识别同目录下的 Excel 源文件
* 100% 保留源表的数据和排版样式
* 根据科目代码和名称，自动将支出明细分流到对应的费用类别（如办公费、印刷费、差旅费等）
* 自动排版、自动计算金额并汇总



## 环境要求

运行此脚本需要 Python 环境，并安装以下依赖库：`openpyxl`



## 安装与运行

1. 克隆或下载本仓库代码到本地
2. 在项目目录中激活 Python 虚拟环境
3. 安装依赖：
   ```bash
   pip install openpyxl
   ```
4. 将需要处理的经费原始表格放入与脚本相同的文件夹内（需要先转换为`.xlsx`格式）
5. 运行脚本
   ```bash
   python process_excel.py
   ```
6. 根据终端提示（若有），程序将自动生成后缀为 `_分类汇总.xlsx` 的结果文件



## 注意事项与数据安全

* 本项目严格遵守数据隔离原则，代码仓库中不包含任何真实的财务数据文件。
* 在使用 Git 管理本项目时，请确保 `.gitignore` 文件正常生效，切勿将含有真实敏感信息的 Excel 表格 Push 到远程仓库。
